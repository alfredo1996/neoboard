## Docker Compose (Recommended)

The fastest way to run NeoBoard with all dependencies:

```bash
git clone https://github.com/alfredo1996/neoboard.git
cd neoboard
cp .env.example .env.local
docker compose up
```

This starts:
- **NeoBoard** on `http://localhost:3000`
- **PostgreSQL** (metadata database) on port 5432
- **Neo4j** (sample database) on ports 7474/7687

## From Source

For development or customization:

```bash
git clone https://github.com/alfredo1996/neoboard.git
cd neoboard

# Start database containers
docker compose up -d postgres neo4j

# Install dependencies
npm ci --prefix connection
npm ci --prefix component
npm ci --prefix app

# Run migrations
npm run db:migrate

# Start dev server
npm run dev
```

The app will be available at `http://localhost:3000`.

## First-Time Setup

1. Navigate to `http://localhost:3000/signup`
2. Create the first admin account
3. The first user automatically gets the **admin** role

## Verify Installation

After signing in, you should see the dashboard list page. If the page loads without errors, NeoBoard is running correctly.
