Get NeoBoard running in minutes with Docker Compose, then create your first dashboard.

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) and Docker Compose
- A Neo4j or PostgreSQL database to connect to (or use the included dev containers)
