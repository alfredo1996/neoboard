## Prerequisites

- **Node.js 20+**
- **Docker** and Docker Compose
- **npm** (not pnpm or yarn)

## Clone and Install

```bash
git clone https://github.com/alfredo1996/neoboard.git
cd neoboard

# Start dev databases
docker compose up -d

# Install all packages
npm ci --prefix connection
npm ci --prefix component
npm ci --prefix app

# Run database migrations
npm run db:migrate

# Start the dev server
npm run dev
```

## Available Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server (port 3000) |
| `npm run build` | Production build + type-check |
| `npm run lint` | ESLint all packages |
| `cd app && npm test` | App unit tests |
| `cd component && npm test` | Component unit tests |
| `cd connection && npm test` | Connection integration tests (needs Docker) |
| `npm run test:e2e` | Playwright E2E tests |
| `npm run storybook` | Component library viewer |
| `npm run db:migrate` | Run Drizzle migrations |
| `npm run db:generate` | Generate migration from schema |

## Environment Setup

Copy `.env.example` to `.env.local` and fill in the required values. See [Configuration](/docs/getting-started/configuration) for details.
